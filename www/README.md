Pre-req:
1. Node.js  
2. xCode or Android Studio (optional)


Install Ionic
```bash
$ npm install -g ionic
```

Navigate to base folder of app

Then run:

```bash
$ ionic emulate ios
```
or

```bash
$ ionic emulate android
```

If you don't have xCode or Android Studio, run the app in the browser by running

```bash
$ ionic serve
```
